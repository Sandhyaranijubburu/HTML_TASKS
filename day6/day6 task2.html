EMMETS AND DESCRIPTION

EMMETS(Name) DESCRIPTION
1.! or (html5)
  • Generates basic thml5 template
  • syntax : html>head>title^body 
    < html>
        < head>
             < title>< /title>
        < /head>
        < body>
        < /body>
        < /html>
    		•
    2.child:>
  • Used to create child elements within the parent element
  • syntax : nav>ul>li
      < nav>
        < ul>
            < li> < /li>
        < /ul>
    < /nav>
    • 3.Sibling:+
  • It is used to create sibling elements
    • syntax : div+p+bq 
    < div >< /div >
        < p >< /p>
        < blockquote>< /blockquote>
    4.Climb-up: ^
  • It is used to moves up one level in hierarchy,allowing us to add sibling elements to the parent elements
  • syntax : div+div>p>span+em^bq 
    < div>< /div>
        < div>
            < p>< span>< /span>< em>< /em p>
            < blockquote>< /blockquote>
        < /div>
  • syntax : div+div>p>span+em^^bq 
    < div> < /div>
        < div>
             < p>< span>< /span>< em>< /em>< /p>
        < /div>
        < blockquote>< /blockquote>
    		•
    5.Multiplication:*

  • It is used to get multiple elements

  • syntax : ul>li*5 
    < ul>
        < li>< /li>
        < li>< /li>
        < li>< /li>
        < li>< /li>
        < li>< /li>
    < /ul>
    		•
    6.Item numbering: $
  • syntax : ul>li.item$*5 
            < ul>
            < li class="item1">< /li>
            < li class="item2">< /li>
            < li class="item3">< /li>
            < li class="item4">< /li>
            < li class="item5">< /li>
        < /ul>
    • 7.Grouping: ()
  • syntax : div>(header>ul>li*2>a)+footer>p 
    < div>
    < header>
        < ul>
            < li>< a href="">< /a>< /li>
            < li>< a href="">< /a>< /li>
        < /ul>
    < /header>
    < footer>
        < p>< /p>
    < /footer>
< /div>
    		•
    8.Text: {}
  • To create text with in the element
  • syntax : a{Click me} 
    < a href="">Click me< /a>
  • syntax :p>{Click }+a{here}+{ to continue} 
    < p>Click < a href="">here< /a> to continue< /p>
    		•
    9.Custom attributes: []
  • To show custom attributes
  • syntax : td[rowspan=2 colspan=3 title] 
    < td rowspan="2" colspan="3" title="">< /td>
